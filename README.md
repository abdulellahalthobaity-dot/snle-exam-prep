## ✨ Key Features

### 📚 Comprehensive Content
- **30 structured lessons** across 5 official SCFHS domains
- **534+ practice questions** with detailed rationales
- Content sourced from **NCLEX-RN Review Made Incredibly Easy** (Lippincott, 2011) and authored content
- Bloom's Taxonomy classification (Application, Analysis, Comprehension, Knowledge)

### 🎯 Mock Exam Engine
- **200 MCQs** in 2 parts (100 each) + 30-min break
- **4 hours 30 minutes** total duration
- **500/800 scaled scoring** (matches official SNLE)
- **Blueprint-proportioned** question distribution:
  - Adult Nursing: 45%
  - Maternity & Women's Health: 18%
  - Pediatric Nursing: 17%
  - Mental Health: 10%
  - Leadership & Informatics: 10%

### 💳 Integrated Payments
- **Salla** electronic payment gateway integration
- 3 pricing tiers:
  - 🎯 Mock Exam (one-time): 100 SR
  - 📚 1 Month Access: 150 SR
  - 🎓 3 Months Access: 300 SR (Best Value)

### 👥 Multi-Role System
- **Student** dashboard with progress tracking, bookmarks, wrong-answer review
- **Admin** panel with code generation, revenue analytics, content management
- **Salla Order Helper** — one-click code issuance from order details

### 📊 Analytics
- Per-student progress dashboards
- Per-domain performance breakdown
- Mock exam history with scaled scores
- Leaderboard

## 🚀 Quick Start

### For Students

1. Visit the platform URL
2. Register as a student (name + ID)
3. Purchase access via Salla (instant secure payment)
4. Send your Salla order number to the admin via WhatsApp
5. Receive your activation code within minutes
6. Enter the code → start studying!

### For Admins

1. Log in with admin code (provided separately)
2. Use **⚡ Salla Order Helper** to:
   - Enter Salla order number + customer name
   - Generate activation code automatically
   - Send pre-filled WhatsApp message in Arabic
